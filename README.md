# Business Sales Performance Analytics

## Project Overview

This project analyzes business sales data using Power BI.

## Objective

- Analyze sales performance
- Identify top-selling products
- Analyze profit by category
- Identify best-performing regions
- Provide business recommendations

## Tools Used

- Power BI
- Microsoft Excel

## Dataset

Superstore Dataset

## Dashboard Features

- Total Sales
- Total Profit
- Total Orders
- Sales Trend
- Category Analysis
- Region Analysis
- Top 10 Products
- Customer Segment Analysis

## Key Insights

- Technology generated the highest revenue.
- West Region had the highest sales.
- Furniture had lower profit margins.
- Sales increased significantly during year-end.

## Recommendations

- Invest more in Technology products.
- Reduce discounts on low-profit products.
- Expand business in high-performing regions.
